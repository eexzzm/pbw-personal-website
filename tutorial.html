1. HTML Introduction

HTML (HyperText Markup Language) is used to create the structure of web pages.

<!DOCTYPE html>
    <html>
    <head>
        <title>Judul Halaman</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
    </body>
    </html>

2. PHP Introduction

PHP is a programming language for creating dynamic web pages.

<?php
        echo "Hello, World!";
    ?>

3. Data type and variable

<?php
        $text = "Hello"; // String
        $number = 10; // Integer
        $float = 10.5; // Float
        $boolean = true; // Boolean
    ?>

4. If and Switch Statement

<?php
        $nilai = 75;
        if ($nilai >= 70) {
            echo "Lulus";
        } else {
            echo "Tidak Lulus";
        }
    ?>
<?php
        $warna = "merah";
        switch ($warna) {
            case "merah":
                echo "Berhenti";
                break;
            case "kuning":
                echo "Hati-hati";
                break;
            default:
                echo "Jalan";
        }
    ?>

5. Basic Loop

<?php
        for ($i = 1; $i <= 5; $i++) {
            echo "Angka: $i \n";
        }
    ?>

6. Array with PHP

<?php
        $buah = ["Apel", "Mangga", "Pisang"];
        foreach ($buah as $b) {
            echo $b . " ";
        }
    ?>
